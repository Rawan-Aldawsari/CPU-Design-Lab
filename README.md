## 🧠 Overview

✨ CPU Design Lab is a smart interactive system that allows users to build their own virtual CPU.

🎯 The system simulates how architectural decisions affect:

| Metric        | Description                          |
|--------------|--------------------------------------|
| ⚡ Performance | Execution speed and efficiency       |
| 🔋 Power      | Energy consumption                   |
| 🌡 Heat       | Thermal output                       |

---

## 🎮 Features

- 🧩 Build your own CPU configuration

## 🧩 How It Works

### 🛠 Step 1: Configure CPU

| Component        | Options                         | Effect |
|-----------------|--------------------------------|--------|
| Registers       | 4 / 8 / 16 / 32                | Faster data access |
| Cache           | None / L1 / L1+L2              | Reduces latency |
| Pipeline        | 3 / 5 / 8 stages               | Improves throughput |
| Clock Speed     | Low / Medium / High            | Affects speed & heat |
| Cores           | 1 / 2 / 4                      | Parallel processing |

## 🔥 Key Concept: Trade-offs

⚖️ No perfect design exists.

| Decision            | Benefit 🔥         | Cost ⚠️              |
|--------------------|------------------|---------------------|
| Increase Cache     | Faster speed     | Higher energy       |
| More Pipeline      | Better throughput| More complexity     |
| Higher Clock       | Faster execution | More heat & power   |
